Events

An event is a signal that something has happened. All DOM nodes generate such signals (but events are not limited to DOM).

1. Mouse events click dblclick mousemove mouseover mousewheel mouseout contextmenu mousedown mouseup 2. Touch events touchstart touchmove touchend touchcancel 3. Keyboard events keydown keypress keyup 4. Form events focus blur change submit 5. Window events scroll resize hashchange load unload
